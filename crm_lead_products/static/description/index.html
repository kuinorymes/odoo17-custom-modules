NetFrame Sync Ukrsibbank

NetFrame Sync Ukrsibbank provides seamless integration between Odoo and Ukrsibbank for automated online bank statement synchronization and Payment creation.

Key Features

  • Automatic fetching of online bank statements from Ukrsibbank
  • Pull statements for custom date ranges on demand
  • Scheduled statement synchronization
  • Creating payments from Odoo
  • Secure connection with OAuth 2.0 & JWT
  • All requests signed with RSA private key

Configuration

Getting Started

To start pulling online bank statements, configure the following in Odoo. First, go to the Accounting app
Create a new Journal of type Bank:

  • In field Account Number insert IBAN of an Account 1
  • Set Bank Feeds to "Online (OCA) 2
  • In section Accounting Information specify a required Currency 3
  • In section Online Bank Statements (OCA) choose Ukrsibbank as Provider 4
  • Save changes 5

After clicking SAVE an Online Bank Statement Provider will be automatically created for your Journal. Now we need to configure it. Go to Settings → Banks → Online Bank Statement Provider:

Select the record named after your Journal ("Ukrsibbank", in our case):

  1. Scheduled update interval. Interval used for scheduled pull. If 1 day was specified - Odoo will connect and try to obtain bank statements once a day. 1
  2. Client ID & Client Secret. Credentials, received from bank after submitting public part of RSA key. 2
  3. Statement Creation Mode. Frequency of statement pulls:
    • Daily statements
    • Weekly statements
    • Monthly statements
    3
  4. Allow Empty Statements. If checked - will create empty bank statements if during integration with bank API no statements were found for specified date. Otherwise - will only create statements for days when there were actual transactions 4
  5. RSA Key (Private part). Field to upload RSA Key (Private part) 5
  6. With Passphrase. Field for passphrase to decode RSA key (private part) 6
  7. After everything is filled click save 7
  8. After saving changes click Connect 8

After Connect button pressed You will be redirected to Ukrsib business. Here You need to login with existing user credentials

Connection in Odoo will change it's status to Connected
Upon successful connection You will be redirected to the following page with status OK:

Close the tab, go back to Odoo and refresh the page. Connection status will change to Token Obtained. This means active connection with Ukrsibbank is established
Now we can try to pull statements from connected Bank Account

Payment Creation

For payment document to be created in Ukrsib Bank the following fields should be filled for the vendor:

  1. In a Contact card specify contact's Name. Use official company name 1
  2. Tax ID: VAT number; EDRPOU for companies and INN for private entrepreneurs (PE) 2
  3. Contact number/Номер контракту: for PE only 3
  4. Bank account: valid IBAN used for transactions with vendor 4
  5. Payment purpose: auto filled field which uses Contact number field and predefined value; can be changed 5

After these steps, return to the Payment Creation section
