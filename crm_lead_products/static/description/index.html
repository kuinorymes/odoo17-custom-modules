E-Chat Integration (Viber, Telegram, WhatsApp)

E-Chat is a platform for integrating Viber, Telegram, and WhatsApp into Odoo.

Centralized management of all connected channels and quick setup via API tokens and webhooks.

Features

  • Automatic lead creation upon the first message
  • Communication history saved in CRM or Discuss
  • Support for incoming/outgoing messages, file attachments, and automatic assignment of conversations
  • Display of name, photo, and channel tags for each contact

Module Functionality

Auto-create leads, manage chats with attachments, reassign managers, switch between messengers, and view channel tags directly in the lead card. All messages appear in the chatter with clear indicators for new ones.

Configuration

Connect a Number in E-Chat

Create an account in E-Chat. Add numbers for Viber, Telegram, and WhatsApp with API integration. Check the connection status under the Integrations menu.

Configuration in Odoo

Go to CRM → Configuration → Social Media and create a connection for each messenger. Fill in the following fields:

  • Name
  • Social Media Type
  • Company
  • Store incoming messages in (CRM Lead or Discuss)
  • Sales Team
  • Echat Access Token
  • Connect Number

Webhooks

In the created connection, click Connect Channel and copy the Callback URL. Paste it into the E-Chat dashboard → Webhooks for both incoming and sent messages.

Testing

Send a message from a connected messenger and check its delivery in Odoo → CRM Lead or Discuss. The module ensures centralized communication, automated lead creation, and efficient distribution.
